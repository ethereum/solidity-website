Solidity v0.8.20

Solidity 0.8.20 includes a range of improvements in the via-IR pipeline and improves the list of events exposed in the contract ABI, and, most importantly, introduces support for the Shanghai hard fork!

We have also included 3 bugfixes in this release!

Solidity is evolving rapidly

We aim for a regular (non-breaking) release every month, with approximately one breaking release per year. You can follow the implementation status of new features in the Solidity Github project.

Stay Updated

As a relatively young language, Solidity is advancing at a rapid speed. We aim for a regular (non-breaking) release every month, with approximately one breaking release per year. You can follow the implementation status of new features in the Solidity Github project.

You can see the upcoming changes for the next breaking release by switching from the default branch (develop) to the breaking branch. You can actively shape Solidity by providing your input and participating in the language design.
